[h1]UIOD Extended Topbar for DLC + MP Spacebar Pause[/h1]

This mod adds spacebar pause support to [b]UI Overhaul Dynamic - Extended Topbar for DLC[/b].

It is intended for players who use UIOD's DLC Extended Topbar variant and want the familiar spacebar pause behavior in multiplayer without manually patching the topbar UI file.

[h2]Features[/h2]
[list]
[*]Adds spacebar pause support to the UIOD Extended Topbar for DLC interface.
[*]Keeps the original UIOD ETFDLC layout and functionality intact.
[*]Generated from upstream UIOD files so updates can be checked and reapplied cleanly.
[*]Designed for Stellaris 4.4.* and UIOD 4.4.*.
[/list]

[h2]Required Mods[/h2]
[list]
[*]UI Overhaul Dynamic
[*]UI Overhaul Dynamic - Extended Topbar for DLC
[/list]

[h2]Load Order[/h2]
Place this mod after both [b]UI Overhaul Dynamic[/b] and [b]UI Overhaul Dynamic - Extended Topbar for DLC[/b].

Recommended order:
[olist]
[*]UI Overhaul Dynamic
[*]UI Overhaul Dynamic - Extended Topbar for DLC
[*]UIOD ETFDLC + MP Spacebar Pause
[/olist]

[h2]Compatibility[/h2]
This mod patches:

[code]interface/main_topbar.gui[/code]

It will conflict with other mods that also replace or heavily modify the same topbar file.
